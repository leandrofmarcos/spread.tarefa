Em função do momento que estou não consegui concluir toda a atividade. 
Pois estava ministrando um treinamento nesta semana, seguido de um momento turbulento na entrega do meu projeto.

- Criei apenas um projeto separando por pacotes para simplificar, mas num cenário real estes seriam divididos em projetos para diminuir acoplamento 
- Não foi concluído todas as regras de operação, mas foi criado uma classe com métodos para a lógica.
- Não foi implementado Injeção de dependência, mas utilizado interface para diminuir o acoplamento.
- Não foi criado tela, mas efetuado testes na classe main bem como nos Testes Unitários
- Foi utilizando um banco in memory (MongoDB) para não comprometer a execução da tarefa por falta de configuração na hora de executar
